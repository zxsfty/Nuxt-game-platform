# 全栈游戏平台项目结构

## 前端 (Nuxt3 + Vue3 + TypeScript)
- `/pages` - 页面路由
  - `index.vue` - 游戏官网首页
  - `games/` - 游戏信息中心
  - `forum/` - 论坛模块
  - `chat/` - 聊天群模块
  - `market/` - 交易平台
  - `launcher/` - 游戏启动器
- `/components` - 公共组件
  - `ThemeToggle.vue` - 主题切换组件
- `/assets` - 静态资源
  - `css/` - 样式文件
- `/stores` - 状态管理 (Pinia)
  - `game.ts` - 游戏相关状态
  - `user.ts` - 用户相关状态
- `/utils` - 工具函数

## 后端 (Node.js + Python + C++)
- `/server` - Node.js主服务
  - `controllers/` - 业务逻辑
  - `routes/` - API路由
  - `middlewares/` - 中间件
  - `services/` - 微服务
- `/game` - C++游戏核心
- `/scripts` - Python辅助脚本

## 数据库 (MySQL)
- 用户系统
- 论坛数据
- 聊天记录
- 交易数据

## 其他
- `docker/` - 容器化配置
- `docs/` - 项目文档
- `.env` - 环境变量
- `nuxt.config.ts` - Nuxt配置文件
- `tailwind.config.js` - TailwindCSS配置文件

- 中国传统节气风格（白露、寒露、小寒）
- 支持主题切换（白露、寒露、小寒）

## 技术栈详情

### 前端技术
- **Nuxt3**: SSR渲染框架，提升首屏加载速度和SEO
- **Vue3**: 组件化开发，使用Composition API
- **TypeScript**: 类型安全，提高代码可维护性
- **TailwindCSS**: 原子化CSS，快速构建UI
- **Pinia**: 状态管理，替代Vuex
- **Socket.io-client**: 实时通信
- **Three.js**: 3D效果展示

### 后端技术
- **Node.js/Express**: 主要API服务
- **Python/FastAPI**: 数据分析和AI服务
- **C++**: 游戏核心逻辑和高性能计算
- **WebSocket**: 实时通信服务
- **Redis**: 缓存和会话管理
- **JWT**: 身份验证
- **GraphQL**: API查询优化

### 数据库设计
- **MySQL**: 关系型数据存储
- **MongoDB**: 非结构化数据存储
- **Elasticsearch**: 全文搜索引擎
- **数据分片**: 高并发访问优化
- **读写分离**: 提高数据库性能

## 系统架构

### 微服务架构
- 用户服务: 处理注册、登录、权限管理
- 内容服务: 管理论坛、聊天等内容
- 交易服务: 处理物品交易和支付
- 游戏服务: 游戏核心逻辑和数据
- 通知服务: 处理系统消息和提醒

### API设计原则
- RESTful API设计
- API版本控制
- 请求限流与熔断
- 统一错误处理
- 完善的API文档

### 数据流向
- 前端 → API网关 → 微服务 → 数据库
- 实时数据: WebSocket → 消息队列 → 微服务
- 异步任务: 事件总线 → 任务队列 → 工作节点
